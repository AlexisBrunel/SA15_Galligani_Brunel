Projet

    Vous trouverez ici tout ce que nous avons fait durant notre projet

    Pour commencer , nous avons crée une librairie .
    Ensuite nous avons commencé le code
    Notre code est consitué de 3 parties :
    -Code 1
    Ce code nous permet de récupérer les informations des parkings de voiture au format xml et au format texte .
    Comment trouver les fichiers ?
    Les fichiers au format xml sont dans l'onglet Programmes .
    Les fichiers au format texte sont dans l'onglet Voiture .
    -Code 2
    Ce code nous permet de récupérer les informations des parkings à vélo au format xml (1 Seul fichier) et au format texte.
    Comment trouver les fichiers ?
    Le fichier xml est dans l'onglet Programmes
    Les fichiers au format texte sont dans l'onglet Vélo
    -Librairie
    Notre librairie se situe dans l'onglet Programmes
    Dans chaque fichier texte vous trouverez un le Nom du parking , le nombre de places totales , le nombre de places libres
    Sur la page d'acceuil vous trouverez notre interprétation et la réponse au projet .
